Tecnologías que ocupa esta plantilla.

strong

Sirve para poner el texto en negrita. Se ocupa cuando se tiene mucho texto, y es importante poder dar énfasis a una parte en concreto, con la etiqueta strong lo podemos hacer.

div


Hace una división dentro del contenido. Es un elemento que es usado mayoritariamente para agrupar otros elementos y actuar como plantilla de otros controles. La etiqueta "div" nos ayuda a estructurar el documento en secciones.

span

Sirve para dar estilos de una parte del texto. Con esta etiqueta podemos personalizar el estilo de solamente una parte del texto.

a

Sirve para crear enlaces. Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar. Otro atributo muy usado es target, el cual nos sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.

p

Etiqueta que da estilos de párrafo. Etiqueta que nos sirve para agrupar texto dentro de un parágrafo. El propósito es poder hacer el contenido más fácil de leer y organizado.
